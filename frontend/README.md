# Frontend (Next.js)

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

## Structure
- `components/` - UI components
- `hooks/` - Custom React hooks
- `pages/` - Next.js pages
- `styles/` - Tailwind and global styles
- `utils/` - Utility functions
- `types/` - TypeScript types
